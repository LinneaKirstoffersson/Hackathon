## Projekt & Teknisk- beskrivning

Vi gör en hämsida för att hjälpa unga att ta sig igenom gymnasiet på ett enkelt och strukturerat sätt. Vi fockuserar på helhetsidén så att hemsidan ska funka ordentligt.  
Vi har byggt en upgifts kalender där studenter kan lägga in skolupgifter så de har god koll på vad de har att göra. Man kan även göra flashcards för att testa sig på sina kunskaper eller för att lära sig saker utantill på ett enkelt sätt.
de olika funktionerna är lösta med javascript som sparas lokalt. 
Funktionen för uppgifts kalendern funkar genom att man väljer ett namn, datum och hur viktigt det är. Sedan sparas det där man kan se sina upgifter. Genom att alla upgifter sparas blir det enklet att se vad man har att göra och vad man ahr satt som viktigt att göra och vad som ska lämnas in först.
Flashcards fungerar lite på samma sätt man skapar olika flashcards sparar dem och sedan kan man använda dem genom att man får up ett och kan kolla på det sedan trycker man på det för att få svaret.
